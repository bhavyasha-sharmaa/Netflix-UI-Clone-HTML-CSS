📺 Netflix UI Clone (HTML & CSS)
This is a simple Netflix-style UI that I built using only HTML and CSS to practice frontend basics.
🚀 What I used
HTML
CSS (Flexbox)
✨ Features
Horizontal scrolling movie cards
Clean layout using flexbox
Image cards with consistent sizing
Hover effect on cards
Responsive-ish design (still improving)
💡 What I learned
How flexbox works in real layouts
Handling overflow and horizontal scrolling
Using object-fit for images
Small CSS properties can affect layout a lot
⚠️ Challenges I faced
Images overflowing in flex container
Aligning cards properly
Scrollbar styling and spacing issues
📌 Note
This is a beginner project and not fully perfect, but I learned a lot while building it. I’ll keep improving it as I learn more.
